Totally-Original-Never-Before-Seen-Scroller

This started as an experiment in writing a java applet just to see how it was done.

This applet can be configured using the following parameters :

  • background colour
  • text colour
  • horizontal/vertical scrolling
  • forward/backward scrolling
  • message by use of an external text file
  • font face and size
  • pause between message repeats
  • pause between lines of text
  • pause when text in centre of screen
  • scrolling speed and smoothness

The code you need to include in your web page is shown below : 


<applet code="ltScroller.class" width = 450 height = 100>
<param name = "bgcolor" value="#FFFFFF">
<param name = "text" value="#000000">
<param name = "horizontal" value = "0">
<param name = "backwards" value = "1">
<param name = "message" value = "demo.txt">
<param name = "font" value = "Times">
<param name = "size" value = "36">
<param name = "repeat" value = "200">
<param name = "line" value = "0">
<param name = "pause" value = "0">
<param name = "delay" value = "50">
<param name = "step" value = "1">
</applet>

You can download a zip file containing the java .class file, the sample html code and a sample message file.
